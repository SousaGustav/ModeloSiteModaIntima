# Aura Intimates

Projeto fictício de uma loja online de moda íntima desenvolvido com foco em design elegante, experiência do usuário e responsividade.

A proposta do projeto foi criar uma landing page moderna para uma marca fictícia de lingeries, buscando transmitir sofisticação, delicadeza e autoestima através da interface visual.

## 🔗 Acesse o projeto:
[Clique aqui!](https://modelo-site-moda-intima.vercel.app/)

### ✨ Funcionalidades
🛍️ Catálogo visual de produtos

Exibição moderna de peças e coleções em destaque.

### 📱 Layout responsivo

Adaptação completa para celulares, tablets e desktops.

### 🎯 Navegação suave

Rolagem fluida entre as seções da página.

### 💬 Botões de contato

Integração visual com WhatsApp para facilitar o contato.

### 📸 Área social

Seção inspirada em integração com Instagram e divulgação visual da marca.

### 💌 Newsletter / Grupo VIP

Área destinada para captação de clientes e promoções.
